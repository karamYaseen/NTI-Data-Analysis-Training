# Problem 2 – Real Estate Database ERD

## Task Description

Prepare an E-R diagram for a real estate firm that lists properties for sale.

• The firm has a number of sales offices in several states. Attributes include Office_Number and Location.

• Each sales office is assigned zero or more employees. Attributes include Employee_ID and Employee_Name.

• An employee must be assigned to only one sales office.

• Each sales office has one manager and manager can’t manage many sales offices at the same time.

• The firm lists property for sale. Attributes include Property_ID and Location (Address, City, State, Zip_Code).

• Each property must be listed with one sales office.

• Each property may have zero or more owners.

• An owner own one or more properties.

• The system stores the percent owned by each owner in each property.
