# Problem 4 – Airline Database ERD

## Task Description

Major airlines companies keep database with information on airlines.

• Each airline has identification number, name, address, contact person and phone numbers.

• Each employee has employee ID, name, address, birth date, gender, position, qualification.

• Each airline owns aircrafts with aircraft ID, capacity, model.

• Aircraft assigned to routes. Information recorded:
  - Number of passengers
  - Price per passenger
  - Departure datetime
  - Arrival datetime
  - Duration

• Each route has route ID, origin, destination, distance, classification.

• Each aircraft has its own crew:
  - Major pilot
  - Assistant pilot
  - Two hostesses

• Each airline keeps transaction data:
  - Transaction ID
  - Date
  - Description
  - Amount
