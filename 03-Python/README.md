# Python — NTI Data Analysis Training

This folder contains the complete Python curriculum for data analysis. All explanations, code examples, and tutorials are integrated into the comprehensive `tutorial.py` file.

## Quick Start

**Run the complete tutorial:**
```bash
python tutorial.py
```

This will execute all course content with detailed comments, generate sample outputs, and create demonstration files.

## Course Map

- **First Day** — [`01-Python-Basics/`](./01-Python-Basics/) — Types, variables, strings, lists, tuples, exercises
- **Second Day** — [`02-Sequences/`](./02-Sequences/) — Sets, dictionaries, control flow, exception handling, exercises
- **Third Day** — [`03-Functions-OOP/`](./03-Functions-OOP/) — Functions, lambdas, classes, OOP, exercises
- **Fourth Day** — [`04-NumPy-Pandas-Plotting/`](./04-NumPy-Pandas-Plotting/) — NumPy, Pandas, Matplotlib, automation sample
- **Session 7 recap** — [`05-Recap-Task/`](./05-Recap-Task/) — SQL Server + Pandas, scraping, regex, automation assignment

## Comprehensive Tutorial

The [`tutorial.py`](./tutorial.py) file contains:
- **Detailed comments** explaining every concept

- **Progressive learning** from basics to advanced
