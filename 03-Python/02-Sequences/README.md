# Python Advanced Topics - Second Day

This section contains advanced Python programming materials covering unordered sequences, control flow, exception handling, and algorithmic problem-solving.

## Topics Covered

### 1. Unordered Sequences
Advanced data structures for unique collections and key-value mappings:
- **Sets**: Unique, unordered collections
  - Creating sets, adding/removing elements
  - Set operations (union, intersection, difference)
- **Dictionaries**: Key-value pair collections
  - Creating and accessing dictionaries
  - Dictionary methods and operations
  - Nested dictionaries

### 2. Control Flow
Program control structures for decision making and iteration:
- **Conditional Statements**: if, elif, else
  - Boolean expressions and truthiness
  - Nested conditions and conditional expressions
- **Loops**: while and for loops
  - Loop control (break, continue, pass)
  - Iterating over sequences

### 3. Exception Handling
Error management and robust programming:
- **Try-Except Blocks**: Catching and handling exceptions
- **Exception Types**: Different types of errors
- **Finally Clause**: Cleanup operations

### 4. Programming Exercises
Practical coding tasks covering:
- **Word Frequency Analysis**: Dictionary operations
- **Prime Number Detection**: Mathematical algorithms
- **Text Processing**: String analysis and palindrome checking
- **List Manipulation**: Element swapping and filtering
- **Error Handling**: Safe user input and file operations
