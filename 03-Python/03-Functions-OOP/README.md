# Python Functions & OOP - Third Day

This section contains Python programming materials covering functions and object-oriented programming (OOP).

## Topics Covered

### 1. Functions
Code organization and reusability:
- **Function Definition**: Creating and calling functions
  - Function syntax and structure
  - Parameters and arguments
- **Parameters**: Different types of parameters
  - Positional arguments
  - Default values
  - Keyword arguments
  - *args (arbitrary positional arguments)
- **Return Values**: Returning data from functions
  - Return statements
  - Type hints and docstrings
- **Advanced Functions**: Lambda and functional programming
  - Lambda expressions
  - map(), filter() functions
  - Scope and global variables

### 2. Object-Oriented Programming (OOP)
Designing programs with classes and objects:
- **Classes and Objects**: Creating blueprints for data
  - Class definition and instantiation
  - Attributes and methods
  - __init__ constructor method
- **Encapsulation**: Data privacy and protection
  - Public and private attributes
  - Properties and getters/setters
  - Data validation
- **Inheritance**: Code reuse through class hierarchy
  - Base classes and subclasses
  - super() function
  - Method overriding
- **Magic Methods**: Special methods for customization
  - __str__, __repr__ methods
  - Operator overloading (__add__, etc.)
- **Practical Applications**: Real-world class design
  - Account management
  - Employee hierarchy
  - Card games

After completing this section, you should be able to:

- Use lambdas and functional programming techniques

- Implement inheritance and method overriding
- Understand and use magic methods
- Build complex programs using OOP principles
