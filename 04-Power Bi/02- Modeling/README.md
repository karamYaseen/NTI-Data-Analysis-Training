# 02 — Data Modeling (E‑commerce sales)

Practice a **sales-oriented** Power BI model: relate fact and dimension tables, write DAX measures, and build a dashboard for e‑commerce-style questions (revenue, orders, trends).

## Typical lab files

These often sit next to this README when you build the module:

| File | Use |
|------|-----|
| `E-Commerce Sales.pbix` | Main report |
| `Sales.xlsx` | Excel source (paths must match the PBIX or refresh will fail) |
| `E-Commerce Sales.png` | Optional dashboard screenshot |

> **Folder name:** This directory is `02- Modeling` (space after `02-`). Keep names consistent with your PBIX data source paths.

## What you practice

- Star/snowflake-style thinking in Power BI (facts vs dimensions)
- Relationships and filter direction
- Measures for KPIs and time intelligence where the lab requires it
