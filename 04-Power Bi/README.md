# Power BI Training Modules

Technical assessment: **Build dashboards**, **model data**, use **DirectQuery**, and perform **churn analysis** through hands-on labs.

---

## Quick start

1. Install [Power BI Desktop](https://powerbi.microsoft.com/desktop/)
2. For Lab 3 and Full Project: Install SQL Server + [SSMS](https://learn.microsoft.com/en-us/sql/ssms/download-sql-server-management-studio-ssms?view=sql-server-ver16)
3. Follow each module's README in order

One run per module produces:

- **PBIX files** — Power BI reports and dashboards
- **Optional exports** — PNG snapshots for portfolio

---

## Scenario

Progressive Power BI training: from basic dashboards to advanced modeling, DirectQuery, and ETL-driven analysis.

---

## Modules

| Module | Topic | Focus |
|--------|--------|-------|
| [`01-Basic-Dashboard/`](01-Basic-Dashboard/) | Kickstarter CSV → dashboard | Data connection, shaping, visuals |
| [`02- Modeling/`](02-%20Modeling/) | E‑commerce star schema + DAX | Data modeling, measures |
| [`03- Direct Query/`](03-%20Direct%20Query/) | Lab 3 — AdventureWorks OLTP | DirectQuery, complex model |
| [`04- Full Project/`](04-%20Full%20Project/) | Churn analysis (SQL ETL + report) | ETL, advanced analytics |

**Naming:** folders use space after number (`02- Modeling`). In URLs use `%20`.

**Scope:** Lab 3 is only `03- Direct Query/`. Capstone: `Final Project/`.

**Tools:** Power BI Desktop; SQL Server + SSMS for database modules.
