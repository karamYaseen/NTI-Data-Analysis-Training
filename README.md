# NTI Data Analysis Training

Personal coursework and portfolio for the **NTI Data Analysis** program — **Karam Yaseen**. This repository is **not** an official NTI release; it is organized so **you** (or future readers) can follow the same path: numbered topics, clear READMEs, and a **highlight capstone** in `06-Final-Project/`.

---

## Repository map

| Folder | Topic |
|--------|--------|
| [`01-Database-Design/`](01-Database-Design/) | ERD and relational mapping practice (case studies + README per problem). |
| [`02-SQL/`](02-SQL/) | SQL labs: create/load data, queries, advanced SQL. |
| [`03-Python/`](03-Python/) | Python track from basics through NumPy, Pandas, and plotting. |
| [`04-Power Bi/`](04-Power%20Bi/README.md) | Power BI: dashboards, modeling, DirectQuery, full lab project. |
| [`05-Excel/`](05-Excel/README.md) | Excel-related exercises and sample data (see folder README). |
| [`06-Final-Project/`](06-Final-Project/README.md) | **Capstone:** e‑commerce analysis — `task.py` + explanatory Jupyter notebook. |

---

## Suggested order

1. Follow the numbered folders: database design → SQL → Python → Power BI.  
2. Use **`05-Excel/`** in parallel or when your schedule aligns with spreadsheet labs.  
3. Treat **`06-Final-Project/`** as the integrative capstone: data prep, analytics, and optional Power BI work.

---

## How to run Python work

| Goal | Command |
|------|--------|
| General Python labs | Create a venv, then `pip install -r requirements.txt` at the **repo root** (see [`03-Python/README.md`](03-Python/README.md)). |
| Capstone only | `cd 06-Final-Project` → `pip install -r requirements.txt` → `python task.py` **or** open `notebooks/final_project_walkthrough.ipynb`. |

Power BI: use **Power BI Desktop**; Lab 3 and similar may use SQL Server — see [`04-Power Bi/README.md`](04-Power%20Bi/README.md).
